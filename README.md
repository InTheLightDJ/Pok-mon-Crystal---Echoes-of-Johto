# Pokémon Essentials GBC

# What is this?
A modified Pokémon Essentials project that allows to create GBC-style Pokémon games.

# Created using
Pokémon Essentials v21.1.

# Current version
v1.0

# How to update your project?
1. Clone this repository to your own Githun Desktop app.
2. Copy the contents of this project and paste them on your root folder.

Make sure you backup your changes before replacing anything, because this will overwrite everything.
